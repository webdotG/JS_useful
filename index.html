JavaScript задачи и примеры

  • Задача:

    Написать функцию которая принимает массив чисел
    и возвращает сумму всех положительных элементов массива

    Данные:

    [-1, 7, 4, -21, 9, 0, -2]

    Решение:

    
function getPositiveFunction(array) {
    let result = 0
    for(let i = 0; i < array.length; i++){
        if(array[i] > 0) {
        result += array[i]
        }
    }
    return result
}

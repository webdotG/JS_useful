## Запуск проекта

Для запуска проекта :

**Установка зависимостей:**
   Введите команду в терминале, чтобы установить зависимости.

   ```bash
   npm install
  ```
**Запуск локального сервера:**
   После успешной установки зависимостей, запустите локальный для просмотра в браузере

   ```
   npm run start
  ```
